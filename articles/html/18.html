    鉴于中国互联网的稳定性，我们不保证此教程在任何时间、地区均可用。
    这篇文章将使用Outline帮助你快速搭建自己的翻墙服务器。

    STEP.0 准备

    1.一个境外的VPS服务器（Vultr、搬瓦工等，购买教程可以去搜索“Vultr 购买教程”），并且服务器的IP未被屏蔽。

    2.一台运行着Windows 7/8/10的PC，且能连接到你的服务器。




    STEP.1 下载Outline Manager 并且安装

    官网链接：点此下载（如果被墙了请使用备用下载地址）


    备用下载：点此下载




    安装方式与安装其他软件相同。




    STEP.2 在服务器上获取Api key

    在服务器登录，并在终端内运行这行命令：curl -sS https://get.docker.com/ | sh
    执行：wget -qO- https://raw.githubusercontent.com/Jigsaw-Code/outline-server/master/src/server_manager/install_scripts/install_server.sh | bash
    终端将输出如下内容，请将其复制备用。

    {

    "apiUrl": "https://xxxxxxxxxxxxxxxxxxxxxxxxx",

    "certSha256": "xxxxxxxxxxxxxxxxxxxxxxxxxxx"

    }




    STEP.3 配置Outline Manager


    1.打开安装好的Outline Manager，界面看起来应该像下面这样：Outline Manager 欢迎界面

    2. 看到窗口右面的滚动条了吗？翻到最底下，然后点一下红圈标注的按钮：Outline Manager 欢迎界面


     
    3. 看到这个页面了吧！在红圈内的输入框 粘贴 刚才从终端复制好的内容，然后点击“DONE”按钮，程序会自动添加服务器信息。Outline Manager 配置服务器页面




    STEP.4 导出SS://链接给客户端使用



    在上一步配置完成后，Outline Manager会自动进入主界面，你的主界面应该看起来是这个样子的：Outline Manager 主界面按照下面各图点击红色框出的按钮



    "GET CONNECTED" 按钮移动上去才会背景变灰
    点击"CONNECT THIS DEVICE"


     
    点击"COPY"来导出SS://链接到剪贴板


    到这里，我们成功导出了SS://链接，接下来继续讲述如何使用SS://链接。




    STEP.5 连接到你的服务器

    下载Shadow Socks客户端

    Android：Outline Android客户端 - Google Play

    iOS & Mac OS X：暂无，Outline支持iOS & Mac OS X后本文会更新。暂时请考虑使用Shadow Rocket代替，关于Shadow Rocket的使用教程还请多多百度。

    Windows：Outline版SS客户端

    我们不保证所有客户端没有后门。

    Outline Manager导出的是正常的SS://链接，理论上支持原版Shadow Socks、Shadow Socks R、Shadow rocket等连接。如果不能链接请尝试启用"UDP 转发"功能。

    2. 打开客户端，导入链接

    Windows 操作方法：添加链接






    点击链接稍等10s左右，程序将自动连接，届时电脑可正常访问国际互联网。

    Android 操作方法：

    1.在手机上复制SS://链接后打开Outline，会自动提示添加，点击"ADD SERVER"即可添加。


    就像这个样子
    2. 添加成功后点击"CONNECT"点击"CONNECT"


     
    3. 检查自己手机的通知栏是否有如下通知，如果有，则恭喜你成功连接到了国际互联网！[删除线]才不是Android P呢╭(╯^╰)╮[/删除线]




    STEP.6 你已连接到国际互联网 & 几句忠告

    恭喜！此篇教程到这里就可以了。但是，请记住如下几点：

    永远不要向不信任的人共享你的SS://链接。记住，某个服务器同时连接的不明流量越多，这个服务器越有可能被墙。
